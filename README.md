**📖 About**

A standalone Windows executable for generating and scanning QR codes. No bloat, no dependencies, no browser required. Works offline for basic tasks. Batch processing and custom styling included.

**⚙️ Requirements**

- Windows 10 / 11 (x64)
- 4GB RAM
- 50MB free disk space
- Internet for updates only
- Webcam (for scanning)

**✨ Features**

**Generation** 🔧 — Create QR codes for URLs, text, Wi-Fi, vCards, and more. Customize colors, sizes, and error correction levels.

**Scanning** 🔍 — Read any QR code from screen, image file, or webcam in real-time. History logs each scan.

**Batch Mode** 📦 — Process hundreds of inputs at once. Export results as CSV or images.

**CLI Support** 💻 — Fully functional from command line. Integrate into scripts or automation pipelines.

**No Cloud** 🛡️ — Everything runs locally. Your data stays on your machine.

**Portable** 💨 — Single .exe file. No install required. Run from USB stick.

**Custom Templates** 🎨 — Save and reuse preset styling for branded QR codes.

**Export Formats** 📤 — PNG, JPG, SVG, PDF. High-res printing support.

**🔧 Configuration**

```json
{
  "output_path": "./output",
  "default_format": "png",
  "error_correction": "M",
  "module_size": 10,
  "color_foreground": "#000000",
  "color_background": "#FFFFFF"
}
```

Config is auto-generated on first run at `%APPDATA%\QRCodeTool\config.json`.

**💻 CLI Usage**

```
qrcode.exe generate --text "https://example.com" --output ./qr.png --size 500
qrcode.exe scan --from ./image.png --output ./result.json
qrcode.exe batch --input ./urls.txt --format svg
qrcode.exe help
```
